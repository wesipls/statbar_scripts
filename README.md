# statbar_scripts
Small C programs for swaybar.
## memory.c
Displays current RAM usage:  
**Output**: `RAM: 8.2G/16G`
## swap.c
Displays current swap usage:  
**Output**: `SWP: 2.5G/8G`
## network_in_out.c
Displays current network usage:  
**Output**:`NET: ↓ 1.5M / ↑ 800K`
## cpu.c
Displays current CPU usage:  
**Output**: `CPU: 15.3%`
